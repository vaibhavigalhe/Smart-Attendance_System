# Smart Attendance System 🎯
## 📌 Description
This is an AI-based Smart Attendance System that uses face recognition to automatically mark attendance using a webcam.
It helps reduce manual attendance and improves accuracy.
---
## 🚀 Features
- Face detection using webcam  
- Real-time face recognition  
- Automatic attendance marking  
- Stores attendance in CSV file  
---
## 🧠 Tech Stack
- Python  
- OpenCV  
- DeepFace  
- Pandas  
---
## ⚙️ How to Run
1. Install required libraries:
pip install opencv-python pandas deepface
2. Capture faces:
python capture_faces.py
3. Run attendance system:
python train_model.py
